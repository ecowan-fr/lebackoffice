Par défaut, le backoffice utilise les timbres de classification d'Ecowan, via le CDN.

Pour utiliser vos propres timbres de classification, ajoutez dans ce dossier les timbres que vous souhaitez remplacer.

Le SVG est obligatoire afin de toujours avoir une qualité optimale.

TYPE : FICHIER
--------------------------------------------------
RESTREINT : restreint_timbre.svg
CONFIDENTIEL : confidentiel_timbre.svg
SECRET : secret_timbre.svg
